# PokéDex Simple JavaScript App

## Description
This is a web application using HTML, CSS, and JavaScript that retrieves data from an external API. It allows user to view details about the original 150 pokémon.

## Purpose of the App
This app was created to learn about programming and complete a project that is a fully functioning JavaScript web application. It relies on an external data source for retrieving information. The learning goal also includes utilizing HTML and CSS in the application.

## Key Features
* Load data from an external source (API)
* View a list of items
* On user action (e.g., by clicking on a list item), view details for that item

## User Stories
* As a user, I want to be able to receive information on a single Pokémon character.
* As a user, I want to be able to see images and details about specific characteristics such as height and abilities.

## Languages and Tech Used
* HTML
* CSS
* JavaScript
* Bootstrap
* jQuery
